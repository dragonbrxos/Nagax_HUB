# NAGAX HUB | Blox Fruits
Script profissional modularizado e 100% funcional.

## Como Executar

### 1. Modo Totalmente Automático (Auto Farm)
Executa toda a linha de raciocínio de farm, itens e eventos sem interface.
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/dragonbrxos/Nagax_HUB/main/Scripts/Auto.lua"))()
```

### 2. Modo Manual (Com Interface)
Abre o painel completo para seleção manual de funções.
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/dragonbrxos/Nagax_HUB/main/Scripts/Manual.lua"))()
```

## Estrutura de Módulos
- **Combat**: Bypass e Fast Attack.
- **RaceV4**: Trials, Mirage Island e Gears.
- **Events**: Sea Beast, Leviathan e Terror Shark.
- **Items**: CDK, Soul Guitar e Puzzles.
- **Farm**: Leveling automático para Mundo 1, 2 e 3.
