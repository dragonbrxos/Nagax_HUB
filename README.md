# NAGAX HUB | Blox Fruits

O script mais roubado e profissional do Blox Fruits, agora totalmente modularizado.

## Estrutura do Projeto
- **Modules/**: Lógica de Farm, Quests e Combate separada.
- **UI/**: Nova Biblioteca de Interface moderna e sem emojis.
- **Scripts/**: Versões Auto e Manual.

## Como Executar

### 1. Versão 100% Automática (Sem GUI)
Ideal para deixar farmando sozinho. Ele mostra o progresso no console (F9).
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/dragonbrxos/Nagax_HUB/main/Scripts/Auto.lua"))()
```

### 2. Versão Manual (HUB com Interface)
Ideal para quem quer selecionar as funções manualmente.
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/dragonbrxos/Nagax_HUB/main/Scripts/Manual.lua"))()
```
