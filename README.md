# Nagax_HUB

Um Hub de Blox Fruits organizado, modular e eficiente.

## Como usar
Para executar o Nagax_HUB, utilize o seguinte script:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/dragonbrxos/Nagax_HUB/main/Main.lua"))()
```

## Estrutura do Projeto
- `Main.lua`: Ponto de entrada principal.
- `Modules/`: Contém lógica compartilhada, como bypass de anti-cheat e utilitários.
- `Tabs/`: Contém a lógica específica para cada aba da interface (Auto Farm, ESP, etc.).
